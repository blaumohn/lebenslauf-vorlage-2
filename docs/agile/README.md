## Konvention
- `ISS-xxx`: umsetzbare Issues im Delivery-Fluss.
- `STY-xxx`: uebergeordnete Story mit mehreren Teil-Issues.
- `BLC-xxx`: fruehe Backlog-Candidates (noch keine konkrete Umsetzung).
